# Webinar: Del Archivo Tradicional al Archivo Inteligente

Landing en HTML para el webinar **“Del Archivo Tradicional al Archivo Inteligente”** del IFAP (Instituto de Formación Archivística Peruana). La página presenta el temario, casos, hoja de ruta y llamado a inscripción.

## Objetivo

Comunicar el valor del archivo inteligente, motivar la inscripción y brindar una experiencia visual clara y atractiva.

## Contenido principal

- Hero con fecha, hora y CTA
- Temario del webinar
- Evolución del archivo y uso de IA
- Automatización, seguridad y normativa
- Casos prácticos
- Hoja de ruta de 90 días
- Rol del archivista
- Recursos y bibliografía
- Cierre con CTA

## Tecnologías

- HTML5 + CSS3 (layout y estilos)
- JavaScript (navegación activa y menú móvil)
- Google Fonts (Karla, Syne)
- Font Awesome (íconos)

## Estructura

- `index.html`: página publicada en GitHub Pages
- `webinar de tema unoi.html`: archivo fuente editable

## Uso local

Abre `index.html` en tu navegador.

## Publicación en GitHub Pages

1. Sube este repositorio a GitHub.
2. Ve a **Settings ? Pages**.
3. En **Build and deployment** selecciona:
   - **Source**: Deploy from a branch
   - **Branch**: `main` (o `master`) y carpeta `/root`
4. Guarda los cambios y espera la URL pública.
