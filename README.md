# Webinar: Del Archivo Tradicional al Inteligente

Este repositorio contiene una landing en HTML para un webinar. Está lista para desplegarse en GitHub Pages.

## Vista previa local

Abre `index.html` en tu navegador.

## GitHub Pages

1. Sube el repositorio a GitHub.
2. Ve a **Settings ? Pages**.
3. En **Build and deployment** selecciona:
   - **Source**: Deploy from a branch
   - **Branch**: `main` (o `master`) y carpeta `/root`
4. Guarda los cambios y espera 1–2 minutos.

La URL pública tendrá el formato:

```
https://TU_USUARIO.github.io/NOMBRE_DEL_REPO/
```

## Archivos

- `index.html`: página principal para GitHub Pages.
- `webinar de tema unoi.html`: archivo original editable.

## Notas

- Si actualizas `webinar de tema unoi.html`, recuerda copiarlo otra vez a `index.html` para reflejarlo en Pages.
